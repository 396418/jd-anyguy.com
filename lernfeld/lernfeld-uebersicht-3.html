Programmierung

Hier vertiefst du dein Verständnis für die reine Logik der Softwareentwicklung. Abseits von bunten Oberflächen lernst du, wie Algorithmen aufgebaut sind, wie Daten im Hintergrund verarbeitet werden und wie man sauberen Code schreibt.

Du arbeitest mit Kontrollstrukturen und Funktionen, um mathematische und logische Probleme automatisiert zu lösen.

Inhalte

  • Datentypen, Variablen und Operatoren
  • Kontrollstrukturen (Verzweigungen & Schleifen)
  • Funktionen, Parameter und Rückgabewerte
  • Arrays und grundlegende Datenstrukturen
  • Fehlersuche (Debugging) & Code-Dokumentation

Lernziele

  1. Programmlogik eigenständig in Code übersetzen
  2. Strukturierte Algorithmen entwickeln
  3. Fehler im Code mittels Debugger finden und beheben
